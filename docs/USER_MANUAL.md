## Getting Started

Code Buddy is an AI-powered programming tutor designed specifically for Indian school students. The platform uses a three-tier access system: **Administrators** set up the system, **Teachers** manage the learning environment, and **Students** learn programming through interactive AI tutoring.

### System Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Stable internet connection
- No software installation required

---

## For Administrators

### Initial Setup

#### Step 1: Admin Login
1. Open Code Buddy in your web browser
2. You'll see the welcome screen with login options
3. Enter the following credentials:
   - **Gemini API Key**: Your Google Gemini API key
   - **Admin Password**: `Skidmin2025`
   - **Terminal Password**: Create a password for teachers (remember this!)
4. Click **"Activate Terminal"**

#### Step 2: API Key Setup
- Get your Gemini API key from [Google AI Studio](https://makersuite.google.com/app/apikey)
- The API key is used to power the AI tutoring features
- Keep your API key secure and don't share it publicly

#### Step 3: Terminal Password
- Create a memorable password for teachers
- Teachers will use this password to access the learning interface
- You can change this password anytime from the admin dashboard

### Admin Dashboard Features

#### API Key Management
- **Status Monitoring**: See if your API key is working correctly
- **Health Check**: Test API connectivity anytime
- **Usage Tracking**: Monitor API usage (in full version)

#### System Status
- **Green**: API key is healthy and working
- **Yellow**: API is throttled (temporary, will resolve)
- **Red**: API key invalid or connection issues

#### Troubleshooting Admin Issues
- **Invalid API Key**: Double-check your Gemini API key
- **Connection Error**: Check internet connection
- **Throttled**: Wait a few minutes, this is normal

## For Teachers

### Accessing the System

#### Step 1: Teacher Login
1. Open Code Buddy
2. Click **"Switch to Teacher Login"** (if not already selected)
3. Enter the **Terminal Password** provided by your administrator
4. Click **"Start Teaching"**

#### Step 2: Starting a Learning Session
1. After login, you'll see the welcome screen
2. Click **"Start Learning"** to begin
3. The system is now ready for students

### Managing Student Learning

#### Grade Selection
Students will first choose their grade level:
- **Junior (Grades 4-6)**: "The Thinker's Path" - Basic computational thinking
- **Explorer (Grades 7-9)**: "The Python Adventure" - Introduction to Python programming
- **Advanced (Grades 10-12)**: "The Builder's Toolkit" - Advanced programming concepts

#### Monitoring Progress
- Students' progress is automatically tracked
- Each topic shows completion status
- Time estimates help plan lessons
- Progress persists across sessions

#### Language Settings
Students can choose their preferred language:
- **Primary Language**: English, Hindi, Tamil, Telugu, Bengali, Marathi
- **Regional Language**: Optional secondary language support
- **Code Highlighting**: Toggle word-by-word code highlighting

### Best Practices for Teachers

#### Classroom Management
1. **One Browser Tab**: Keep Code Buddy open in one tab during class
2. **Student Guidance**: Help students with grade selection initially
3. **Progress Monitoring**: Check curriculum maps to see student progress
4. **Break Management**: Students can pause and resume anytime

#### Encouraging Students
- The AI tutor uses encouraging Hinglish phrases
- Students receive immediate feedback on their code
- Celebrate completions with the class
- Use the cultural analogies to reinforce learning

## For Students

### Starting Your Learning Journey

#### Step 1: Choose Your Grade
When you first start, select your grade level:
- **Junior**: If you're in grades 4-6
- **Explorer**: If you're in grades 7-9  
- **Advanced**: If you're in grades 10-12

#### Step 2: Meet Your AI Tutor
- Your AI tutor is like a friendly computer wizard
- It will explain concepts using familiar examples (like making chai or cricket teams)
- Ask questions anytime - your tutor loves to help!

### Learning with Code Buddy

#### Understanding the Interface
- **Chat Area**: Where you talk with your AI tutor
- **Input Box**: Type your messages and code here
- **Action Buttons**: Quick responses like "I got it!" or "Explain differently"
- **Settings**: Change language preferences

#### Writing and Running Code
1. **Listen to Explanation**: Your tutor will explain a concept first
2. **Get a Challenge**: You'll receive a coding task
3. **Write Code**: Type your code in a code block:
   ```python
   print("Hello, World!")
   // run
   ```
4. **Add // run**: This tells the system to execute your code
5. **Get Feedback**: Immediate results and guidance

#### Action Buttons
After explanations, you'll see helpful buttons:
- **"Yep, I got it!"**: When you understand
- **"Explain it differently"**: If you need another explanation
- **"I have a question..."**: To ask specific questions
- **"Show me an example"**: For concrete examples

### Learning Tips for Students

#### Getting the Most from Your Tutor
1. **Ask Questions**: Your AI tutor loves questions!
2. **Try Your Best**: Don't worry about mistakes - they help you learn
3. **Use Examples**: The tutor uses familiar Indian examples
4. **Take Breaks**: Learning is better in small chunks
5. **Practice**: Try variations of the code you learn

#### Understanding Feedback
- **Green Messages**: Success! You got it right
- **Helpful Hints**: When code needs improvement, you get specific guidance
- **No Direct Answers**: The tutor guides you to discover solutions yourself

#### Progress Tracking
- **Curriculum Map**: Shows all topics and your progress
- **Completed Topics**: Marked with green checkmarks
- **Current Topic**: Highlighted in blue
- **Upcoming Topics**: Shown in gray

## Troubleshooting

### Common Issues and Solutions

#### Login Problems
**Problem**: Can't log in as teacher
- **Solution**: Check with your administrator for the correct terminal password
- **Note**: Password is case-sensitive

**Problem**: Admin login not working
- **Solution**: Verify admin password is `Skidmin2025`

#### Learning Interface Issues
**Problem**: Code won't run
- **Solution**: Make sure you add `// run` at the end of your code block
- **Example**:
  ```python
  print("Hello")
  // run
  ```

**Problem**: AI tutor not responding
- **Solution**: Check internet connection, refresh the page if needed

**Problem**: Lost progress
- **Solution**: Progress is saved automatically, but refreshing might clear recent chat

#### Technical Issues
**Problem**: Page won't load
- **Solution**: 
  1. Check internet connection
  2. Try refreshing the page
  3. Clear browser cache
  4. Try a different browser

**Problem**: Slow responses
- **Solution**: This is normal during high usage, responses may take a few seconds

### Getting Help
1. **Refresh First**: Many issues resolve with a page refresh
2. **Check Internet**: Ensure stable connection
3. **Ask Teacher**: Teachers can help with learning-related questions
4. **Contact Admin**: For technical issues, contact your system administrator

## FAQ

### General Questions

**Q: Do I need to install anything?**
A: No! Code Buddy runs entirely in your web browser.

**Q: Can I use this on my phone?**
A: Yes, Code Buddy works on phones, tablets, and computers.

**Q: Is my progress saved?**
A: Yes, your progress is automatically saved and will be there when you return.

### Learning Questions

**Q: What if I don't understand something?**
A: Use the "Explain it differently" button or ask specific questions. Your AI tutor has many ways to explain concepts!

**Q: Can I go back to previous topics?**
A: Currently, the system progresses forward through the curriculum. Focus on understanding each topic before moving on.

**Q: What programming language will I learn?**
A: You'll primarily learn Python, which is perfect for beginners and very powerful.

### Technical Questions

**Q: Why does my code sometimes not work?**
A: This is part of learning! The AI tutor will give you hints to fix your code. Debugging is an important programming skill.

**Q: Can I change the language?**
A: Yes! Click the settings button to choose from English, Hindi, Tamil, Telugu, Bengali, or Marathi.

**Q: What if the AI tutor makes a mistake?**
A: AI tutors are very smart but not perfect. If something seems wrong, ask for clarification or try refreshing the page.

### Administrative Questions

**Q: How much does this cost?**
A: The cost depends on API usage. Administrators manage the API key and associated costs.

**Q: Can multiple students use this at once?**
A: Yes! Each student gets their own learning session and progress tracking.

**Q: Is this suitable for my grade level?**
A: Code Buddy has three levels designed for grades 4-12. Choose the level that matches your current grade.
